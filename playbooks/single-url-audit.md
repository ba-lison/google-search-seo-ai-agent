# Single URL Audit Playbook

Use when the user gives one page or a small URL set.

## Required Spec

- URL;
- goal for the URL;
- intended audience;
- target query/topic if known;
- available HTML/rendered content.

## Process

1. Check technical eligibility.
2. Check visible content and intent match.
3. Check AI Search readiness.
4. Check structured data if applicable.
5. Provide one-page action plan.

## Output

```text
URL:
Goal:
Readiness: strong | moderate | weak
Must fix:
Should improve:
AI/Search implications:
Validation:
```
