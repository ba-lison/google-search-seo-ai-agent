# Full Site Audit Playbook

Use for domain-level or multi-template audits.

## Required Spec

- site/domain;
- business type;
- priority templates or sections;
- data available;
- desired report audience.

## Subagents

Run:

1. technical SEO;
2. content quality;
3. AI search readiness;
4. structured data;
5. measurement if data exists;
6. synthesis.

## Process

1. Create audit spec.
2. Inventory important sections and URL types.
3. Delegate technical access and indexing review.
4. Delegate content and AI readiness review by template.
5. Delegate structured-data review for eligible page types.
6. Delegate measurement review if Search Console/Analytics are available.
7. Consolidate into executive report and backlog.

## Output

- executive summary;
- critical blockers;
- opportunities;
- dev backlog;
- marketing/content backlog;
- validation plan;
- 30/60/90 day roadmap.
